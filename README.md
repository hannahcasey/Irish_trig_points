# Irish trig point map

This repository provides an interactive map of triangulation (trig) points across Ireland.

**Please note:** This project is a work in progress. The map is not yet complete and some trig points are missing or may have inaccurate details.

## Acknowledgements

Most of the underlying trig point data was sourced from information on DeformedWeb, which contains geographic information compiled by members of Trigonomy by vidually scanning Discoverer/y maps.

The original dataset can be found at:
http://www.deformedweb.co.uk/trigs/
